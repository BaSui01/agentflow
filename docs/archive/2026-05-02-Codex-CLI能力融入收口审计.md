## 1. 目标重述

本轮目标不是“模仿 Codex CLI 的 TUI”，而是把 OpenAI 官方最新 Codex CLI 的高价值 runtime semantics 收口进 AgentFlow 当前架构，使其能够更完整地调用模型、工具、MCP、多代理协作与执行策略处理问题。

本轮实际收口的能力维度包括：

- 模型 catalog / provider 参数映射
- approval policy / sandbox mode
- memory external-context policy
- subagent / handoff / fan-out policy
- request entrypoint -> RunConfig -> ExecutionOptions 的统一 override 链
- 中文文档、计划治理与最终收口记录
