## 3. 统一目标架构（修订）

保持现有分层不变，仅做执行口径统一：

1. 统一执行链：`cmd/agentflow/main.runServe -> internal/app/bootstrap.InitializeServeRuntime -> cmd/agentflow/server_*.Start -> bootstrap.RegisterHTTPRoutes -> api/routes -> api/handlers -> domain(agent/rag/workflow/llm)`。
2. 统一入口：
   - LLM：`llm/gateway`
   - Agent：`agent/runtime.Builder`
   - Workflow：`workflow/facade.ExecuteDAG`（对外唯一）
   - RAG：`rag/runtime.Builder + rag/retrieval/pipeline`
3. 统一出口：
   - 错误：`types.ErrorCode`
   - 观测：`trace_id/run_id/parent_run_id` + 节点维度 + usage/cost/retrieval 指标。
4. 统一守卫：
   - 依赖方向：`go-arch-lint`
   - 自定义约束（预算、结构一致性）：`architecture_guard_test.go`
   - 本地脚本：`arch_guard.ps1` 只做 orchestration，不重复实现同类规则。
