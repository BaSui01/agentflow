# ⚡ 五分钟快速开始

> 从零到运行你的第一个 AgentFlow 程序，只需 5 分钟！

## 🎯 本文目标

在本教程结束时，你将：
- ✅ 安装 AgentFlow
- ✅ 配置 API Key
- ✅ 运行第一个对话程序
- ✅ 理解基本工作流程

## 📋 前置条件

- Go 1.24 或更高版本
- 任意 LLM API Key（OpenAI / Claude / DeepSeek / 通义千问 等）

## 🚀 Step 1: 安装

```bash
# 创建新项目
mkdir my-agent && cd my-agent
go mod init my-agent

# 安装 AgentFlow
go get github.com/BaSui01/agentflow
```

## 🔑 Step 2: 配置 API Key

创建 `.env` 文件：

```bash
# OpenAI
OPENAI_API_KEY=sk-xxx

# 或者使用国产模型
# DEEPSEEK_API_KEY=sk-xxx
# QWEN_API_KEY=sk-xxx
```

## 📝 Step 3: 编写代码

创建 `main.go`：

```go
package main

import (
    "context"
    "fmt"
    "os"

    "github.com/BaSui01/agentflow/llm"
    "github.com/BaSui01/agentflow/llm/providers"
    "github.com/BaSui01/agentflow/llm/providers/openai"
    "go.uber.org/zap"
)

func main() {
    // 1. 创建 Logger
    logger, _ := zap.NewDevelopment()
    defer logger.Sync()

    // 2. 创建 Provider
    provider := openai.NewOpenAIProvider(providers.OpenAIConfig{
        BaseProviderConfig: providers.BaseProviderConfig{
            APIKey:  os.Getenv("OPENAI_API_KEY"),
            BaseURL: "https://api.openai.com",
            Model:   "gpt-4o-mini",
        },
    }, logger)

    // 3. 发起对话
    resp, err := provider.Completion(context.Background(), &llm.ChatRequest{
        Model: "gpt-4o-mini",
        Messages: []llm.Message{
            {Role: llm.RoleUser, Content: "用一句话介绍 Go 语言"},
        },
    })
    if err != nil {
        panic(err)
    }

    // 4. 输出结果
    fmt.Println("🤖 AI:", resp.Choices[0].Message.Content)
}
```

## ▶️ Step 4: 运行

```bash
# 加载环境变量并运行
export $(cat .env | xargs) && go run main.go
```

**预期输出：**
```
🤖 AI: Go 是一门由 Google 开发的静态类型、编译型编程语言，以简洁、高效和强大的并发支持著称。
```

你已经成功运行了第一个 AgentFlow 程序！

<summary>Q: 报错 "API key is required"</summary>

echo $OPENAI_API_KEY  # 应该显示你的 API Key
