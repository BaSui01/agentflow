# 多模态处理

AgentFlow 提供完整的多模态内容处理能力，支持图像、音频、视频的输入理解与生成。

> 建议先读：
> - 近 12 个月官方主流模型总表：`../guides/近12个月主流多模态模型总表.md`
> - 统一模型与媒体端点参考：`../guides/模型与媒体端点参考.md`
> - 当前代码已实现多模态能力矩阵：`../guides/多模态能力端点参考.md`

---

## 近 12 个月推荐示例模型速览

| 模态 | 推荐示例模型 | 说明 |
|---|---|---|
| 图像生成 | `gpt-image-1`、`imagen-4.0-generate-001` | 当前教程示例优先用 OpenAI / Gemini 这两类主流家族 |
| 视频分析 | `gemini-2.5-pro`、`gemini-3.1-pro-preview` | 视频理解优先展示 Gemini 家族 |
| 视频生成 | `veo-3.1-generate-preview`、`gen4_turbo` | Google Veo / Runway Gen-4.5 是较新主流示例 |
| 语音合成 (TTS) | `gpt-4o-mini-tts`、`eleven_v3` | OpenAI / ElevenLabs |
| 语音转文本 (STT) | `gpt-4o-transcribe`、`nova-3` | OpenAI / Deepgram |
| Embedding | `text-embedding-3-small`、`gemini-embedding-001` | 文档检索/向量化常用示例 |

## 图像处理

### 图像输入

```go
// 从 URL 创建图像内容
imageContent := multimodal.NewImageURLContent("https://example.com/image.png")

// 从 Base64 创建
imageContent := multimodal.NewImageBase64Content(base64Data, multimodal.ImageFormatPNG)

// 从文件加载
imageContent, err := multimodal.LoadImageFromFile("./image.png")

// 从 URL 下载并加载
imageContent, err := multimodal.LoadImageFromURL("https://example.com/image.png")
```

### 图像生成：OpenAI `gpt-image-1`

```go
import "github.com/BaSui01/agentflow/llm/capabilities/image"

provider := image.NewOpenAIProvider(image.OpenAIConfig{
    APIKey: os.Getenv("OPENAI_API_KEY"),
})

resp, err := provider.Generate(ctx, &image.GenerateRequest{
    Prompt:         "一只可爱的猫咪在阳光下睡觉",
    NegativePrompt: "模糊, 低质量",
    Model:          "gpt-image-1",
    N:              1,
    Size:           "1024x1024",
    Quality:        "high",
    ResponseFormat: "url",
})

for _, img := range resp.Images {
    fmt.Printf("图像 URL: %s\n", img.URL)
}
```

### 图像生成：Gemini / Imagen 4

```go
geminiProvider := image.NewGeminiProvider(image.GeminiConfig{
```

## 音频处理

### 语音合成 (TTS)：OpenAI `gpt-4o-mini-tts`

```go
import speech "github.com/BaSui01/agentflow/llm/capabilities/audio"

ttsProvider := speech.NewOpenAITTSProvider(speech.OpenAITTSConfig{
    APIKey: os.Getenv("OPENAI_API_KEY"),
})

resp, err := ttsProvider.Synthesize(ctx, &speech.TTSRequest{
    Text:           "你好，欢迎使用 AgentFlow！",
    Model:          "gpt-4o-mini-tts",
    Voice:          "alloy",
    Speed:          1.0,
    ResponseFormat: "mp3",
})

audioData, _ := io.ReadAll(resp.Audio)
_ = audioData
```

### ElevenLabs TTS：`eleven_v3`

```go
elevenProvider := speech.NewElevenLabsProvider(speech.ElevenLabsConfig{
    APIKey: os.Getenv("ELEVENLABS_API_KEY"),
})

resp, err := elevenProvider.Synthesize(ctx, &speech.TTSRequest{
    Text:     "高质量语音合成",
    Model:    "eleven_v3",
    Voice:    "Rachel",
    Language: "zh-CN",
})
```

### 语音识别 (STT)：OpenAI `gpt-4o-transcribe`

```go
sttProvider := speech.NewOpenAISTTProvider(speech.OpenAISTTConfig{
    APIKey: os.Getenv("OPENAI_API_KEY"),
})

resp, err := sttProvider.Transcribe(ctx, &speech.STTRequest{
    Audio:                  audioReader,
    Model:                  "gpt-4o-transcribe",
    Language:               "zh",
    ResponseFormat:         "verbose_json",
    TimestampGranularities: []string{"word", "segment"},
})

fmt.Printf("转录文本: %s\n", resp.Text)
```

### Deepgram STT：`nova-3`

```go
deepgramProvider := speech.NewDeepgramProvider(speech.DeepgramConfig{
    APIKey: os.Getenv("DEEPGRAM_API_KEY"),
})

resp, err := deepgramProvider.Transcribe(ctx, &speech.STTRequest{
    Audio:       audioReader,
    Model:       "nova-3",
    Language:    "zh",
    Diarization: true,
})

for _, seg := range resp.Segments {
    fmt.Printf("[%s] %s\n", seg.Speaker, seg.Text)
}
```

## 多模态消息

构建包含多种内容类型的消息：

```go
msg := multimodal.MultimodalMessage{
    Role: "user",
    Contents: []multimodal.Content{
        multimodal.NewTextContent("请分析这张图片中的内容"),
        multimodal.NewImageURLContent("https://example.com/image.png"),
    },
}

// 添加音频
if audioContent, err := multimodal.LoadAudioFromFile("audio.mp3"); err == nil {
    msg.Contents = append(msg.Contents, audioContent)
}
```

## 最佳实践

1. **示例模型优先用近 12 个月主流家族**，但不要把示例模型和项目 fallback 混为一谈。
2. **图像 / 视频大多成本高且耗时长**，要设置合理超时与异步轮询策略。
3. **Gemini 2.5 / 3.x thinking 字段不同**，不要在业务层手写底层 payload。
4. **TTS / STT 优先用支持更好中文与低延迟的模型家族**，例如 `gpt-4o-mini-tts`、`gpt-4o-transcribe`、`nova-3`。
5. **若需要更完整的主流模型列表**，始终回到 `../guides/近12个月主流多模态模型总表.md` 查看。
