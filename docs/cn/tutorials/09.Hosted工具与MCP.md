# Hosted 工具与 MCP

AgentFlow 提供 Hosted 工具，包括文件操作、Shell 命令和 MCP Client，可在工作流中直接调用。

## 文件操作工具

支持 `read_file`、`write_file`、`edit_file`、`list_directory` 四个工具。

### 安全配置

- **AllowedPaths**：白名单路径，仅允许访问指定目录及其子目录；空表示不限制（不推荐生产环境使用）
- **MaxFileSize**：单文件最大字节数，默认 10MB

### 使用示例

通过 workflow DSL 的 `type: tool` 步骤调用：

```yaml
steps:
  read_config:
    type: tool
    tool: read_file
    config:
      path: "/data/config.json"

  list_data:
    type: tool
    tool: list_directory
    config:
      path: "/data"
      recursive: false
```

## Shell 命令工具

`run_command` 用于执行 Shell 命令。

### 安全配置

- **BlockedCmds**：命令黑名单，默认包含 `rm -rf`、`format`、`mkfs` 等危险命令
- **AllowedCmds**：命令白名单，非空时仅允许以白名单前缀开头的命令
- **Timeout**：执行超时，默认 30s，最大 300s

### 使用示例

```yaml
steps:
  run_script:
    type: tool
    tool: run_command
    config:
      command: "go build -o app ./cmd"
      working_directory: "/project"
      timeout_seconds: 60
```

## MCP Client

连接外部 MCP Server，动态发现并调用其提供的工具。

### StdioTransport（子进程模式）

通过标准输入/输出与子进程通信：

```go
import (
    "github.com/BaSui01/agentflow/agent/hosted"
    "github.com/BaSui01/agentflow/agent/protocol/mcp"
)

transport, err := mcp.NewStdioTransport("npx", "-y", "@modelcontextprotocol/server-filesystem", "/data")
if err != nil {
    return err
}
client := mcp.NewDefaultMCPClient(transport, logger)
// Initialize + 注册工具
err = hosted.RegisterMCPTools(ctx, mcp.AsMCPClientLike(client), registry)
```

### SSETransport（HTTP 模式）

通过 SSE 连接远程 MCP Server：

```go
transport := mcp.NewSSETransport("https://mcp.example.com/sse", mcp.WithAuthToken("token"))
client := mcp.NewDefaultMCPClient(transport, logger)
err = hosted.RegisterMCPTools(ctx, mcp.AsMCPClientLike(client), registry)
```

### 动态工具发现：RegisterMCPTools

`RegisterMCPTools` 会调用 MCP Server 的 `tools/list`，将返回的工具逐一注册到 `ToolRegistry`，后续可在工作流中按工具名调用。

## 配置参考

`config.yaml` 中的 `hosted_tools` 部分：

```yaml
hosted_tools:
  file_ops:
    enabled: true
    allowed_paths:
      - /data
      - /tmp/agentflow
    max_file_size: 10485760  # 10MB

  shell:
    enabled: true
    timeout: 30s
    blocked_cmds:
      - "rm -rf"
      - "format "
      - "mkfs"

  mcp:
    enabled: true
    command: "npx"
    args:
      - "-y"
      - "@modelcontextprotocol/server-filesystem"
      - "/data"
    # 或使用 base_url 连接 HTTP SSE 模式
    # base_url: "https://mcp.example.com/sse"
```
