# 成本追踪

AgentFlow 内置 CostTracker，用于记录和聚合 LLM 调用成本。

## CostTracker 架构

```
CostCalculator → CostTracker → CostTrackerLedger → Gateway
```

- **CostCalculator**：按 Provider/Model 计算单次调用的 token 成本
- **CostTracker**：内存存储，记录每次调用并支持按维度聚合
- **CostTrackerLedger**：实现 `Ledger` 接口，Gateway 出口落账时写入 CostTracker
- **Gateway**：在每次 LLM 调用完成后，将 usage/cost 写入 Ledger

## API Endpoints

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/v1/cost/summary | 获取成本汇总（按 Provider、Model、Agent 聚合） |
| GET | /api/v1/cost/records | 获取成本记录列表 |
| POST | /api/v1/cost/reset | 重置成本记录 |

## 按维度聚合

CostTracker 支持以下聚合方法：

- `TotalCost()` — 总成本
- `CostByProvider()` — 按 Provider
- `CostByModel()` — 按 Provider:Model
- `CostByAgent()` — 按 Agent
- `CostBySession()` — 按 Session
- `CostByTool()` — 按工具

## 配置

成本追踪随 Gateway 自动生效，无需额外配置。Ledger 在 bootstrap 阶段注入为 `CostTrackerLedger`，Gateway 出口统一落账。
