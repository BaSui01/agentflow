## Recommended Example Models (Last 12 Months)

| Modality | Recommended example models | Why these examples |
|---|---|---|
| Image generation | `gpt-image-1`, `imagen-4.0-generate-001` | Good default examples for current mainstream OpenAI / Google image families |
| Video analysis | `gemini-2.5-pro`, `gemini-3.1-pro-preview` | Gemini remains the most straightforward video-understanding example in the current project |
| Video generation | `veo-3.1-generate-preview`, `gen4_turbo` | Good examples for Google Veo and Runway Gen-4.5 families |
| TTS | `gpt-4o-mini-tts`, `eleven_v3` | Current mainstream examples for low-latency TTS and premium TTS |
| STT | `gpt-4o-transcribe`, `nova-3` | Good examples for general STT and production speech recognition |
| Embedding | `text-embedding-3-small`, `gemini-embedding-001` | Reasonable defaults for retrieval and vectorization demos |

## Image Processing

### Image Input

```go
// From URL
imageContent := multimodal.NewImageURLContent("https://example.com/image.png")

// From Base64
imageContent := multimodal.NewImageBase64Content(base64Data, multimodal.ImageFormatPNG)

// From file
imageContent, err := multimodal.LoadImageFromFile("./image.png")

// From URL download
imageContent, err := multimodal.LoadImageFromURL("https://example.com/image.png")
```

### Image Generation: OpenAI `gpt-image-1`

```go
import "github.com/BaSui01/agentflow/llm/capabilities/image"

provider := image.NewOpenAIProvider(image.OpenAIConfig{
    APIKey: os.Getenv("OPENAI_API_KEY"),
})

resp, err := provider.Generate(ctx, &image.GenerateRequest{
    Prompt:         "A cute cat sleeping in sunlight",
    NegativePrompt: "blurry, low quality",
    Model:          "gpt-image-1",
    N:              1,
    Size:           "1024x1024",
    Quality:        "high",
    ResponseFormat: "url",
})

for _, img := range resp.Images {
    fmt.Printf("Image URL: %s\n", img.URL)
}
```

### Image Generation: Gemini / Imagen 4

```go
geminiProvider := image.NewGeminiProvider(image.GeminiConfig{
```

## Audio Processing

### Text-to-Speech: OpenAI `gpt-4o-mini-tts`

```go
import speech "github.com/BaSui01/agentflow/llm/capabilities/audio"

ttsProvider := speech.NewOpenAITTSProvider(speech.OpenAITTSConfig{
    APIKey: os.Getenv("OPENAI_API_KEY"),
})

resp, err := ttsProvider.Synthesize(ctx, &speech.TTSRequest{
    Text:           "Hello, welcome to AgentFlow!",
    Model:          "gpt-4o-mini-tts",
    Voice:          "alloy",
    Speed:          1.0,
    ResponseFormat: "mp3",
})

audioData, _ := io.ReadAll(resp.Audio)
_ = audioData
```

### Text-to-Speech: ElevenLabs `eleven_v3`

```go
elevenProvider := speech.NewElevenLabsProvider(speech.ElevenLabsConfig{
    APIKey: os.Getenv("ELEVENLABS_API_KEY"),
})

resp, err := elevenProvider.Synthesize(ctx, &speech.TTSRequest{
    Text:     "High-fidelity speech synthesis",
    Model:    "eleven_v3",
    Voice:    "Rachel",
    Language: "en-US",
})
```

### Speech-to-Text: OpenAI `gpt-4o-transcribe`

```go
sttProvider := speech.NewOpenAISTTProvider(speech.OpenAISTTConfig{
    APIKey: os.Getenv("OPENAI_API_KEY"),
})

resp, err := sttProvider.Transcribe(ctx, &speech.STTRequest{
    Audio:                  audioReader,
    Model:                  "gpt-4o-transcribe",
    Language:               "en",
    ResponseFormat:         "verbose_json",
    TimestampGranularities: []string{"word", "segment"},
})

fmt.Printf("Transcript: %s\n", resp.Text)
```

### Speech-to-Text: Deepgram `nova-3`

```go
deepgramProvider := speech.NewDeepgramProvider(speech.DeepgramConfig{
    APIKey: os.Getenv("DEEPGRAM_API_KEY"),
})

resp, err := deepgramProvider.Transcribe(ctx, &speech.STTRequest{
    Audio:       audioReader,
    Model:       "nova-3",
    Language:    "en",
    Diarization: true,
})

for _, seg := range resp.Segments {
    fmt.Printf("[%s] %s\n", seg.Speaker, seg.Text)
}
```

## Video Processing

### Video Analysis: Gemini `gemini-2.5-pro`

```go
import "github.com/BaSui01/agentflow/llm/capabilities/video"

geminiVideo := video.NewGeminiProvider(video.GeminiConfig{
    APIKey: os.Getenv("GEMINI_API_KEY"),
}, logger)

resp, err := geminiVideo.Analyze(ctx, &video.AnalyzeRequest{
    VideoURL:  "https://example.com/video.mp4",
    Prompt:    "Describe the video content and major actions",

    MaxFrames: 10,
    Interval:  2.0,
```

## Multimodal Messages

```go
msg := multimodal.MultimodalMessage{
    Role: "user",
    Contents: []multimodal.Content{
        multimodal.NewTextContent("Analyze this image"),
        multimodal.NewImageURLContent("https://example.com/image.png"),
    },
}

if audioContent, err := multimodal.LoadAudioFromFile("audio.mp3"); err == nil {
    msg.Contents = append(msg.Contents, audioContent)
}
```

## Best Practices

1. Prefer **date-scoped mainstream model families** in examples, but do not confuse examples with runtime fallback models.
2. Image / video APIs are costlier and slower; set explicit timeouts and polling expectations.
3. Gemini 2.5 and 3.x do not use the same thinking fields; keep provider mapping in the provider layer.
4. For speech flows, modern examples like `gpt-4o-mini-tts`, `gpt-4o-transcribe`, and `nova-3` are better defaults than older families.
5. For a fuller official snapshot, always return to [`../guides/RecentModelFamiliesAndModalities.md`](../guides/RecentModelFamiliesAndModalities.md).
