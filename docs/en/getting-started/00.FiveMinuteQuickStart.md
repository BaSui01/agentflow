# ⚡ Five-Minute Quick Start

> From zero to running your first AgentFlow program in just 5 minutes!

## 🎯 Goals

By the end of this tutorial, you will:
- ✅ Install AgentFlow
- ✅ Configure API Key
- ✅ Run your first chat program
- ✅ Understand the basic workflow

## 📋 Prerequisites

- Go 1.24 or higher
- Any LLM API Key (OpenAI / Anthropic Claude / DeepSeek / Qwen, etc.)

## 🚀 Step 1: Installation

```bash
# Create a new project
mkdir my-agent && cd my-agent
go mod init my-agent

# Install AgentFlow
go get github.com/BaSui01/agentflow
```

## 🔑 Step 2: Configure API Key

Create a `.env` file:

```bash
# OpenAI
OPENAI_API_KEY=sk-xxx

# Or use other providers
# DEEPSEEK_API_KEY=sk-xxx
# QWEN_API_KEY=sk-xxx
# ANTHROPIC_API_KEY=sk-ant-xxx
```

## 📝 Step 3: Write Code

Create `main.go`:

```go
package main

import (
    "context"
    "fmt"
    "os"

    "github.com/BaSui01/agentflow/llm"
    "github.com/BaSui01/agentflow/llm/providers"
    "github.com/BaSui01/agentflow/llm/providers/openai"
    "go.uber.org/zap"
)

func main() {
    // 1. Create Logger
    logger, _ := zap.NewDevelopment()
    defer logger.Sync()

    // 2. Create Provider
    provider := openai.NewOpenAIProvider(providers.OpenAIConfig{
        BaseProviderConfig: providers.BaseProviderConfig{
            APIKey:  os.Getenv("OPENAI_API_KEY"),
            BaseURL: "https://api.openai.com",
            Model:   "gpt-4o-mini",
        },
    }, logger)

    // 3. Send a chat request
    resp, err := provider.Completion(context.Background(), &llm.ChatRequest{
        Model: "gpt-4o-mini",
        Messages: []llm.Message{
            {Role: llm.RoleUser, Content: "Describe Go programming language in one sentence"},
        },
    })
    if err != nil {
        panic(err)
    }

    // 4. Print the result
    fmt.Println("🤖 AI:", resp.Choices[0].Message.Content)
}
```

## ▶️ Step 4: Run

```bash
# Load environment variables and run
export $(cat .env | xargs) && go run main.go
```

**Expected output:**
```
🤖 AI: Go is a statically typed, compiled programming language developed by Google, known for its simplicity, efficiency, and powerful concurrency support.
```

## 🎉 Congratulations!

You've successfully run your first AgentFlow program!

## 💡 FAQ

<details>
<summary>Q: Error "API key is required"</summary>

Make sure the environment variable is set correctly:
```bash
echo $OPENAI_API_KEY  # Should display your API Key
```
</details>

<details>
<summary>Q: How to use DeepSeek instead of OpenAI?</summary>

```go
import "github.com/BaSui01/agentflow/llm/providers/vendor"

provider, err := vendor.NewChatProviderFromConfig("deepseek", vendor.ChatProviderConfig{
    APIKey: os.Getenv("DEEPSEEK_API_KEY"),
    Model:  "deepseek-chat",
}, logger)
if err != nil {
    panic(err)
}
```
</details>

<details>
<summary>Q: How to use Anthropic Claude?</summary>

```go
import "github.com/BaSui01/agentflow/llm/providers/anthropic"

provider := anthropic.NewAnthropicProvider(providers.ClaudeConfig{
    BaseProviderConfig: providers.BaseProviderConfig{
        APIKey: os.Getenv("ANTHROPIC_API_KEY"),
        Model:  "claude-3-5-sonnet-20241022",
    },
}, logger)
```
</details>
