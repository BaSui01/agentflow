# 📦 Installation & Setup

> Detailed installation steps and configuration options

## 📋 System Requirements

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| Go | 1.24.0 | 1.24+ |
| OS | Linux / macOS / Windows | - |
| Memory | 512MB | 2GB+ |

## 🔧 Installation Methods

### Method 1: Go Modules (Recommended)

```bash
# In your project
go get github.com/BaSui01/agentflow
```

### Method 2: Install from Source

```bash
git clone https://github.com/BaSui01/agentflow.git
cd agentflow
go build ./...
```

### Method 3: Specific Version

```bash
# Install a specific version
go get github.com/BaSui01/agentflow@v1.0.0

# Install latest development version
go get github.com/BaSui01/agentflow@latest
```

## 🔑 API Key Configuration

AgentFlow supports 13+ LLM providers. Here's how to configure each:

### Environment Variables (Recommended)

Create a `.env` file:

```bash
# ============ International Providers ============
# OpenAI
OPENAI_API_KEY=sk-xxx
OPENAI_BASE_URL=https://api.openai.com

# Anthropic Claude
ANTHROPIC_API_KEY=sk-ant-xxx

# Google Gemini
GEMINI_API_KEY=xxx

# Mistral
MISTRAL_API_KEY=xxx

# xAI Grok
GROK_API_KEY=xxx

# ============ Chinese Providers ============
# DeepSeek
DEEPSEEK_API_KEY=sk-xxx

# Qwen
QWEN_API_KEY=sk-xxx

# Zhipu GLM
GLM_API_KEY=xxx

# Doubao (ByteDance / Volcano Engine)
DOUBAO_API_KEY=xxx

# Kimi (Moonshot)
KIMI_API_KEY=xxx

# Tencent Hunyuan
HUNYUAN_SECRET_ID=xxx
HUNYUAN_SECRET_KEY=xxx

# MiniMax
MINIMAX_API_KEY=xxx
MINIMAX_GROUP_ID=xxx
```

### Configuration in Code

```go
import (
    "github.com/BaSui01/agentflow/llm/providers"
    "github.com/BaSui01/agentflow/llm/providers/openai"
)

// Method 1: Direct configuration
provider := openai.NewOpenAIProvider(providers.OpenAIConfig{
    BaseProviderConfig: providers.BaseProviderConfig{
        APIKey:  os.Getenv("OPENAI_API_KEY"),
        BaseURL: "https://api.openai.com",
        Model:   "gpt-4o",
    },
}, logger)

// Method 2: Read from environment variables
provider := openai.NewOpenAIProvider(providers.OpenAIConfig{
    BaseProviderConfig: providers.BaseProviderConfig{
        APIKey:  os.Getenv("OPENAI_API_KEY"),
        BaseURL: os.Getenv("OPENAI_BASE_URL"),
        Model:   "gpt-4o",
    },
}, logger)
```

## ⚙️ Advanced Configuration

### Timeout Configuration

```go
import "time"

config := providers.OpenAIConfig{
    BaseProviderConfig: providers.BaseProviderConfig{
        APIKey:  os.Getenv("OPENAI_API_KEY"),
        Timeout: 60 * time.Second,  // Request timeout
    },
}
```

### Proxy Configuration

```go
config := providers.OpenAIConfig{
    BaseProviderConfig: providers.BaseProviderConfig{
        APIKey: os.Getenv("OPENAI_API_KEY"),
    },
}
```

### Custom Base URL (Compatible APIs)

```go
// Use OpenAI-compatible third-party services
config := providers.OpenAIConfig{
    BaseProviderConfig: providers.BaseProviderConfig{
        APIKey:  os.Getenv("API_KEY"),
        BaseURL: "https://your-proxy.com/v1",  // Custom endpoint
        Model:   "gpt-4o",
    },
}
```

## 🔍 Verify Installation

Create `verify.go` to verify the installation:

```go
package main

import (
    "context"
    "fmt"

    agentruntime "github.com/BaSui01/agentflow/agent/execution/runtime"
    "github.com/BaSui01/agentflow/sdk"
    workflowruntime "github.com/BaSui01/agentflow/workflow/runtime"
)

func main() {
    _ = sdk.New(sdk.Options{})
    _ = agentruntime.DefaultBuildOptions()
    _ = workflowruntime.NewBuilder(nil, nil)
    _ = context.Background()

    fmt.Println("✅ sdk unified entry loaded successfully")
    fmt.Println("✅ agent runtime submodule loaded successfully")
    fmt.Println("✅ workflow runtime submodule loaded successfully")
    fmt.Println("🎉 AgentFlow installation verified!")
}
```

Run:
```bash
go run verify.go
```

## 🐛 Common Installation Issues

### Issue 1: Go Version Too Low

```
error: module requires Go >= 1.24
```

**Solution:**
```bash
# Check Go version
go version

# Upgrade Go (macOS)
brew upgrade go

# Upgrade Go (Linux)
sudo rm -rf /usr/local/go
wget https://go.dev/dl/go1.24.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.24.linux-amd64.tar.gz

# Upgrade Go (Windows)
# Download from https://go.dev/dl/ and run the installer
```

### Issue 2: Network Issues

```
go: github.com/BaSui01/agentflow: connection refused
```

**Solution:**
```bash
# Set Go proxy (for users in China)
go env -w GOPROXY=https://goproxy.cn,direct

# Or use other proxies
go env -w GOPROXY=https://goproxy.io,direct
```

### Issue 3: Dependency Conflicts

```
go: inconsistent vendoring
```

**Solution:**
```bash
# Clean and re-download dependencies
go clean -modcache
go mod tidy
go mod download
```
