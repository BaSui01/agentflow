# Engineering Skills

日常编码核心技能。用于代码开发、调试、架构改进。

| 技能 | 触发场景 | 核心价值 |
|------|---------|---------|
| [diagnose](diagnose.md) | 报告 Bug、性能回归 | 系统化调试循环，避免乱猜 |
| [grill-with-docs](grill-with-docs.md) | 设计方案评估、术语精炼 | 对齐理解，产出 CONTEXT.md + ADR |
| [improve-codebase-architecture](improve-codebase-architecture.md) | 代码质量差、难以测试 | 发现深化模块的机会 |
| [prototype](prototype.md) | 设计验证、快速试错 | 一次性可运行原型 |
| [setup-agentflow-skills](setup-agentflow-skills.md) | 首次使用本技能集 | 配置 Issue 追踪器和标签 |
| [tdd](tdd.md) | 新功能开发、Bug 修复 | 红-绿-重构，短反馈循环 |
| [to-issues](to-issues.md) | 计划/PRD 需要拆分任务 | 垂直切片拆解为可执行 Issue |
| [to-prd](to-prd.md) | 已有讨论产出需要归档 | 合成 PRD 并发布 |
| [triage](triage.md) | Issue 需要分类处理 | 状态机驱动的分类流程 |
| [zoom-out](zoom-out.md) | 遇到不熟悉的代码 | 高层架构视角解释 |
